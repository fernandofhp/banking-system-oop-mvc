# 💰 Megabank - Sistema Bancário em Python (MVC + OOP)

Sistema bancário desenvolvido em Python utilizando boas práticas de arquitetura, com separação em camadas (MVC), orientação a objetos e interface em terminal com formatação profissional.

---

## 🚀 Funcionalidades

- ✅ Criação de clientes
- ✅ Criação de contas
- ✅ Depósitos
- ✅ Saques com regras:
  - Limite de valor
  - Limite de quantidade de saques
- ✅ Extrato detalhado:
  - Histórico de transações
  - Valores alinhados
  - Cores (entrada, saída e saldo)
- ✅ Listagem de contas

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (OOP)
- Arquitetura MVC
- Encapsulamento e abstração
- Separação de responsabilidades
- Boas práticas de organização de código

---

## 🖥️ Interface

Sistema executado via terminal com:

- Layout alinhado
- Uso de cores ANSI
- Formatação estilo "sistema bancário real"
